🚀 RoleForge — by NeuralForge

Turn your current skills into a realistic path to your dream role.

⸻

🧠 Overview

RoleForge is a career simulation engine that helps users understand:
	•	Where they stand today
	•	What skills they’re missing
	•	What is realistically achievable given their time and constraints

Unlike typical course recommenders, RoleForge answers:

“Given your current situation, what is the fastest realistic path to become job-ready?”

⸻

✨ Key Features

🔍 Skill Gap Analysis
	•	Extracts user skills (manual input or CV)
	•	Compares against target role requirements
	•	Identifies missing skills with priority ranking

📊 Readiness Score
	•	Calculates current job readiness (%)
	•	Explains strengths and weaknesses

⏱️ Career Simulation
	•	Projects progress over time
	•	Shows realistic improvement based on:
	•	hours per week
	•	learning pace
	•	Displays max achievable readiness

🧭 Smart Roadmap
	•	Generates a 4-week learning plan
	•	Adapts based on:
	•	FastTrack ⚡
	•	BudgetWise 💰
	•	Balanced ⚖️

🛠️ Portfolio Project Generator
	•	Suggests a real-world project
	•	Helps users prove their skills

🔁 Alternative Roles
	•	Recommends adjacent career paths
	•	Helps users pivot smarter

⸻

🏗️ Tech Stack
	•	Frontend & App: Streamlit
	•	Data Processing: Pandas, NumPy
	•	Visualization: Plotly
	•	Data Source: O*NET Database
	•	Optional AI: LLM for roadmap & explanations
